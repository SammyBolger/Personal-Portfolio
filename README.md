# Personal Portfolio

**Live Website:** [sammybolger.com](https://sammybolger.com)

A modern personal portfolio website showcasing my all my work and projects, and a bit about me!. Designed and built from scratch using HTML, CSS, and JavaScript with a focus on clean UI, responsive design, and subtle interactive animations.

## Features

- **Animated typewriter hero section** with dynamic role cycling
- **Responsive two-column layout** with image card
- **Gradient typography styling** for modern visual emphasis
- **Light/Dark mode toggle** with smooth transitions
- **Multi-page navigation** (Home, About, Skills, Projects, Contact)
- **Scroll reveal animations**
- **Mobile-friendly responsive design**
- **Custom footer branding**
- **Social links integration** (GitHub, LinkedIn, X, Instagram)

---

## Site Structure

- **Home** — Animated hero introduction and professional summary
- **About** — Academic background, career goals, and personal interests
- **Skills** — Technical stack and core competencies
- **Projects** — Featured data science and cybersecurity work
- **Contact** — Direct links and contact information

---

## Technologies Used

- **HTML5** for structure
- **CSS3** with custom gradients and responsive layouts
- **Vanilla JavaScript** for interactivity and animations
- **CSS Grid & Flexbox** for layout control
- **Font Awesome** for iconography
- **Google Fonts** for typography

---

## How to Run

Clone the repo:

```bash
git clone https://github.com/SammyBolger/portfolio.git
cd portfolio
```

Open index.html in your browser
